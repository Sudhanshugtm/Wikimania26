# Prototype handoff note design

The QR code points to one stable landing page. The page gives Wikimania attendees brief context and instructions before they eventually continue to a prototype hosted on Test Wikipedia.

The first release uses a single responsive card. It identifies Wikimania 2026, explains that the prototype is not yet available, and presents a visibly disabled call to action. Once the Test Wikipedia URL is known, the call to action becomes an ordinary link without changing the QR destination.

The page must remain readable at 360 px and desktop widths, preserve a comfortable line length, expose semantic landmarks, and avoid representing the unavailable destination as interactive.
